Installation Instructions

To use these cards in Home Assistant:

# Via HACS (Recommended)
1. Go to HACS ‚Üí Frontend
2. Click menu ‚Üí Custom repositories
3. Add: https://github.com/loryanstrant/ha-MU-TH-UR-6000-cards
4. Category: Lovelace
5. Install "MU/TH/UR 6000 Cards"

# Manual Installation
1. Download mu-th-ur-6000-cards.js
2. Copy to config/www/
3. Add resource in configuration.yaml:
   lovelace:
     resources:
       - url: /local/mu-th-ur-6000-cards.js
         type: module

> Example Configuration

# Status Card Example
